## 🚀 Quick Start Installation

This script handles all dependencies, configurations, and service setups automatically.

1.  **Download the Installer:**
    ```bash
    wget https://raw.githubusercontent.com/Panda1847/NullSec-RedTeam-AI/main/install.sh
    ```

2.  **Run the Script (Requires `sudo`):**
    ```bash
    chmod +x install.sh
    sudo ./install.sh
    ```

## 🛡️ Failsafe & Diagnostics (The Guardian Tool)

The **Guardian Tool** is embedded within the installer and deployed to `/usr/local/bin/guardian`. It provides self-healing capabilities:

| Command | Description |
| :--- | :--- |
| `sudo guardian` | Runs a full system integrity check with Pacman animations. |
| `sudo guardian "error message"` | Attempts to diagnose and fix a specific error message automatically. |
